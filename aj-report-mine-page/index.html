功能特点

强大的数据可视化能力，满足企业级需求

📈

丰富图表组件

支持折线图、柱状图、饼图、地图等数十种可视化图表，满足各种数据展示场景

🎨

拖拽式设计

可视化设计器，通过拖拽即可完成大屏布局，无需编写代码，快速搭建数据大屏

🔗

多数据源支持

支持 MySQL、Oracle、PostgreSQL 等主流数据库，以及 RESTful API 数据接入

📱

响应式布局

自适应不同屏幕尺寸，支持大屏投放、PC端、移动端等多种展示场景

⚡

实时数据更新

支持数据定时刷新和 WebSocket 实时推送，确保数据展示的及时性

🛡️

权限管理

完善的用户权限体系，支持多角色管理，保障数据安全
