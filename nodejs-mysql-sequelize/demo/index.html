第一章：项目介绍

了解项目架构与运行原理

项目架构

本项目由两个子模块组成：前端可视化界面（Vue3）和 后端服务（Node + WebSocket）。前端提供图形化配置界面，用户在浏览器中完成数据库连接、表结构设计、视图设置等操作后，后端通过 WebSocket 实时通信，自动生成完整的 Sequelize 后端接口项目。

Sequelize ORM

第二章：环境搭建

启动前端和后端开发服务器

1

启动前端项目

克隆代码后，进入 creat_api_pro_front 目录，执行 npm i 安装依赖，然后 npm run dev 启动开发服务器。

终端 
cd ./creat_api_pro_front
npm i
npm run dev
localhost:5173
前端开发服务器启动成功
2

启动后端服务

在另一个终端窗口中，进入 creat_api_pro_server 目录，同样执行 npm i 安装依赖，然后 node ./server.js 启动后端服务。

终端 
cd ./creat_api_pro_server
npm i
node ./server.js
终端 - node server.js
后端服务启动成功
💡 提示：需要同时运行两个终端窗口，一个跑前端，一个跑后端
