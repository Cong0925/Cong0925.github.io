jQuery SVG D3
流程图编辑器

基于 jQuery、SVG 和 D3.js 构建的交互式流程图编辑器。 支持多种图形形状、节点拖拽、智能连线、力导向图布局、 样式自定义编辑、导入导出等功能，帮助你快速构建专业流程图。

核心功能

强大而直观的 编辑能力

从基础的节点操作到高级的力导向图布局，满足你的各种流程图编辑需求

节点操作

支持矩形、圆形、菱形、平行四边形、椭圆、六边形等多种图形形状。支持拖拽移动、框选、复制粘贴、删除等操作。

  • 多种图形形状
  • 拖拽移动节点
  • 框选批量操作

连线功能

从节点边缘拖拽即可创建连线。支持多种连线样式、颜色自定义、箭头方向控制等。

  • 拖拽创建连线
  • 连线样式编辑
  • 箭头方向控制

力导向图

集成 D3.js 力导向图引擎，自动布局节点，通过物理模拟实现智能排列。

  • D3.js 引擎驱动
  • 自动智能布局
  • 物理模拟效果

样式编辑

所见即所得的样式编辑面板，实时调整节点和连线的填充颜色、边框、字体、阴影等视觉样式。

  • 填充颜色编辑
  • 边框与阴影设置
  • 字体样式调整

导入导出

支持导出为 PNG、SVG、JSON 多种格式，也可导入 JSON 文件恢复编辑状态。

  • PNG / SVG 导出
  • JSON 数据导出
  • JSON 文件导入

数据持久化

支持撤销重做、网格吸附、缩放平移等编辑辅助功能。数据自动保存。

  • 撤销重做操作
  • 网格吸附功能
  • 缩放与平移
技术栈

成熟可靠的 技术组合

jQuery

快速简洁的 DOM 操作与事件处理，提供强大的选择器和链式调用，简化前端开发。

SVG

基于 XML 的矢量图形格式，支持无损缩放、丰富图形元素和交互事件，是流程图的理想载体。

D3.js

Data-Driven Documents，业界领先的可视化库，提供力导向图、数据绑定等高级功能。
