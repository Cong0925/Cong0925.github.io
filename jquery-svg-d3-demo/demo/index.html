快速开始

jQuery SVG D3 流程图编辑器

一款基于 jQuery、SVG 和 D3.js 构建的交互式流程图编辑器。 本教程将帮助你快速安装并掌握编辑器的各项功能。

环境要求

现代浏览器

Chrome、Firefox、Edge、Safari 等现代浏览器（需要支持 ES6+）

本地服务器

需要通过本地服务器访问（不能直接双击 HTML 文件打开）

克隆项目

Terminal 
git clone https://github.com/Cong0925/jquery-svg-d3-demo.git
cd jquery-svg-d3-demo

启动项目

以下三种方式任选其一即可：

1. 使用 VS Code 打开项目文件夹

2. 安装 Live Server 扩展

3. 右键点击 index.html 文件，选择 Open with Live Server

Terminal 
python -m http.server 8080

然后在浏览器中打开 http://localhost:8080

Terminal 
npx serve .

然后在浏览器中打开显示的地址（通常是 http://localhost:3000）

步骤 1

添加节点

编辑器支持多种方式添加节点到画布中。

方式一：从形状库添加

从形状库拖拽添加节点
  1. 在左侧形状库中找到需要的图形
  2. 点击并拖拽到画布区域
  3. 释放鼠标即可添加节点

方式二：双击画布添加

  1. 在画布空白区域双击鼠标
  2. 自动在双击位置添加默认矩形节点
  3. 节点自动进入编辑状态，可直接输入文本
步骤 2

移动节点

通过拖拽可以自由移动节点位置。

点击选中 → 拖拽移动
点击选中并拖拽移动节点
  1. 选中节点：单击画布中的节点
  2. 拖拽移动：按住鼠标左键拖动节点到目标位置
  3. 释放定位：松开鼠标完成移动
提示：开启网格吸附后，节点移动会自动对齐到最近的网格点，便于排版整齐。
步骤 3

复制粘贴节点

使用快捷键快速复制节点，提高编辑效率。

Ctrl + C 复制选中的节点
Ctrl + V 粘贴复制的节点
  1. 选中一个或多个节点
  2. 按下 Ctrl + C 复制
  3. 按下 Ctrl + V 粘贴
  4. 新节点会出现在原节点附近
说明：复制粘贴会保留节点的所有属性，包括颜色、大小、文本内容等。
步骤 4

删除节点

选中节点后按 Delete 键即可删除。

Delete 删除选中的节点
  1. 选中要删除的节点（支持多选）
  2. 按下键盘上的 Delete 键
  3. 节点将被删除，相连的连线也会一并删除
注意：删除操作可以通过 Ctrl + Z 撤销恢复。
步骤 5

添加连线

从节点边缘拖拽可以创建连线，连接两个节点。

从节点边缘拖拽创建连线
  1. 将鼠标移动到节点边缘，光标变为十字形
  2. 按住鼠标左键并拖拽
  3. 拖拽到目标节点边缘释放
  4. 连线自动创建，带有箭头指示方向
提示：连线起点和终点会自动吸附到节点的最近边缘点，确保连线整齐美观。
步骤 6

编辑连线样式

选中连线后可以在样式面板中修改其外观。

  1. 单击选中连线
  2. 在右侧面板中找到连线样式设置
  3. 可修改的样式属性：
    • 线条颜色
    • 线条宽度
    • 线条样式（实线、虚线、点线）
    • 箭头样式
步骤 7

使用对齐功能

多选节点后可以一键对齐，让排版更整齐。

Ctrl + 单击 多选节点
  1. 按住 Ctrl 键，逐个点击要对齐的节点
  2. 点击工具栏中的对齐按钮
  3. 支持的对齐方式：
    • 左对齐
    • 右对齐
    • 顶部对齐
    • 底部对齐
    • 水平居中
    • 垂直居中
提示：也可以使用框选（在画布空白区域按住拖拽）来批量选中节点。
步骤 8

缩放和平移

通过鼠标操作可以自由缩放和移动画布视图。

鼠标滚轮 缩放画布
右键拖拽 平移画布
  1. 缩放：将鼠标放在目标位置，滚动滚轮即可缩放
  2. 平移：按住鼠标右键拖拽，可移动整个画布视图
  3. 适应窗口：使用恢复默认尺寸按钮可快速回到初始视图
步骤 9

恢复默认尺寸

当画布缩放或平移后，可一键恢复到默认视图。

  1. 点击工具栏中的 恢复默认尺寸 按钮
  2. 画布将自动缩放和平移，使所有内容适配窗口显示
提示：此功能在内容较多或缩放比例过大/过小时特别有用。
步骤 10

导出文件

支持将编辑器内容导出为多种格式。

PNG

导出为高清位图格式，适合分享和演示使用

SVG

导出为矢量图形格式，支持无损缩放，适合二次编辑

JSON

导出为结构化数据格式，方便数据交换和后续导入

  1. 点击工具栏中的 导出 按钮
  2. 在下拉菜单中选择导出格式
  3. 选择保存位置并确认
步骤 11

导入 JSON

导入之前导出的 JSON 文件，恢复编辑状态。

  1. 点击工具栏中的 导入 按钮
  2. 选择之前导出的 JSON 文件
  3. 编辑器将自动加载并恢复所有节点和连线
说明：导入 JSON 文件会替换当前画布中的所有内容。建议先导出当前内容作为备份。
