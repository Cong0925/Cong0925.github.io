# Write Helper 写作助手 - 使用演示

这是 Write Helper 软件的图文演示文档，帮助用户快速上手使用。

## 文档结构

```
demo/
├── index.html              # 主页面
├── css/
│   └── style.css           # 样式文件
├── js/
│   └── main.js             # 脚本文件
└── screenshots/            # 截图文件夹
    ├── Snipaste_*.png      # 各步骤截图
```

## 内容概览

1. **下载安装** - 从官网下载并安装软件
2. **初始设置** - 用户须知、设置笔名、领养宠物
3. **项目管理** - 新建项目、选择类型、填写信息
4. **编辑功能** - 编辑器界面、字体设置、功能模块
5. **更多功能** - 小游戏、称号系统、数据统计等

## 使用方法

直接在浏览器中打开 `index.html` 即可查看演示文档。

## 设计风格

- 采用温暖的黄色主题，与 Write Helper 品牌色调一致
- 响应式设计，支持移动端访问
- 图文并茂，每个步骤都有详细截图和说明
