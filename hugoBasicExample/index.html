Hugo 基础示例

基于 Hugo Clarity 主题的个人博客系统，支持中英文双语、登录认证、文章发布， 以极简的黑白金色调呈现优雅的阅读体验。

02

技术架构

基于成熟技术栈构建，确保性能与稳定性

本项目采用 Hugo Extended 作为核心构建引擎，搭配 Hugo Clarity 主题， 实现了现代化的博客体验。前端使用 Glassmorphism 设计风格， 后端通过 JSEncrypt 实现 RSA 密码加密，确保用户认证的安全性。

整个系统基于 Go 语言开发的 Hugo 引擎，编译后的静态文件可部署到任何 Web 服务器， 无需后端运行时环境，极大简化了部署和运维成本。

03

项目结构

hugoBasicExample/
├── assets/css/           # 预编译的主题 CSS
├── config/               # 配置文件
│   └── _default/
│       ├── config.toml   # 主配置文件
│       ├── params.toml   # 网站参数配置
│       └── menus/        # 菜单配置
├── content/              # 内容目录
│   └── post/             # 文章目录
├── layouts/partials/     # 页面模板
│   ├── header.html       # 页头（含登录按钮）
│   ├── login.html        # 登录弹窗
│   └── sidebar.html      # 侧边栏
├── static/               # 静态资源
│   ├── css/              # 自定义样式
│   └── js/               # 自定义脚本
└── themes/               # Hugo 主题
04

快速开始

三步即可启动你的 Hugo 博客

1

安装 Hugo

# Windows
choco install hugo-extended

# macOS
brew install hugo

# Linux
sudo snap install hugo
2

克隆项目

git clone https://github.com/
  Cong0925/hugoBasicExample.git
cd hugoBasicExample
3

启动服务器

hugo server

# 访问 http://localhost:1313/
# 即可预览你的博客
