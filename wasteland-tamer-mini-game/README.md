## 项目简介

《废土中的驯兽师》是一款末日抓宠题材的角色扮演小游戏。玩家将穿越荒芜的废土，捕捉并驯养各异形态的变异生物，通过回合制战斗不断提升实力，挑战区域首领，最终揭开废土沦陷的真相。

## 游戏特色

- **废土世界观**：末日荒芜的废土世界，充满未知与危险
- **变异生物**：各种形态各异的变异生物，等待玩家捕捉
- **回合制战斗**：经典的回合制战斗系统，策略性十足
- **区域探索**：探索不同区域，挑战区域首领
- **剧情推进**：揭开废土沦陷的真相
