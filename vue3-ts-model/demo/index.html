环境准备

1.1 检查 Node.js 版本

Vue3 需要 Node.js >= 16.0.0，推荐使用 18.x 或 20.x 版本。

终端命令
# 检查 Node.js 版本
node -v

# 检查 npm 版本
npm -v

# 检查 Git 版本
git --version
提示：如果未安装 Node.js，请访问 https://nodejs.org/ 下载并安装。

1.2 安装代码编辑器（推荐 VS Code）

推荐使用 VS Code 作为代码编辑器，并安装以下插件：

  • Volar - Vue3 官方插件
  • ESLint - 代码检查
  • Prettier - 代码格式化

克隆项目

2.1 克隆仓库

打开终端，执行以下命令克隆项目：

终端命令
# 克隆仓库
git clone https://github.com/Cong0925/vue3-ts-model.git

# 进入项目目录
cd vue3-ts-model
3

安装依赖

3.1 安装项目依赖

根据你使用的包管理器，选择对应的命令：

npm 安装
npm install
yarn 安装
yarn install
pnpm 安装
pnpm install
预计时间：1-3 分钟（取决于网络速度）
4

启动项目

4.1 启动开发服务器

安装完成后，执行以下命令启动开发服务器：

终端命令
# 启动开发环境
npm run dev

# 或
yarn dev

启动成功后，浏览器会自动打开 http://localhost:8888

成功标志：看到登录页面即表示启动成功

4.2 登录系统

构建部署

5.1 构建生产版本

开发完成后，执行以下命令构建生产版本：

终端命令
# 构建生产版本
npm run build

# 或
yarn build
输出目录：构建产物将输出到 dist/ 目录

5.2 Nginx 部署配置

nginx.conf
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass https://your-api-server.com;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

常见问题

Q1: 安装依赖失败怎么办？

终端命令
# 清除缓存
npm cache clean --force

# 删除 node_modules 重新安装
rm -rf node_modules package-lock.json
npm install

Q2: 端口被占用怎么办？

终端命令
# 查找占用端口的进程 (Windows)
netstat -ano | findstr :8888

# 结束进程 (替换为实际 PID)
taskkill /PID <pid> /F

Q3: 样式不生效怎么办？

  • 确认使用了 <style scoped>
  • 检查 CSS 类名是否正确
  • 确认样式文件已正确导入
  • 重启开发服务器
