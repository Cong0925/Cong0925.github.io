功能特性

用户登录

支持用户名密码登录验证，本地存储用户信息，保持登录状态

实时通信

基于WebSocket协议，实现客户端与服务器的全双工实时通信

多人聊天

支持多个用户同时在线聊天，消息实时广播给所有连接的客户端

前后端分离

Vue3前端 + Node.js后端，清晰的项目结构，易于理解和扩展

响应式设计

适配桌面端和移动端，随时随地进行聊天

快速部署

简单的项目结构，几分钟即可完成本地部署和运行

技术栈

Vue3

渐进式JavaScript框架，使用Composition API和TypeScript

Node.js

JavaScript运行时环境，用于搭建WebSocket服务器

WebSocket

全双工通信协议，实现实时双向数据传输

TypeScript

JavaScript的超集，提供静态类型检查和更好的IDE支持

项目结构

server/

Node.js WebSocket服务器，负责处理客户端连接和消息广播

后端

src/

Vue3前端源码，包含视图、组件、路由、配置等模块

前端

server/index.js

WebSocket服务器入口文件，监听8000端口，处理连接和消息

src/views/login/

登录页面组件，支持用户名密码验证和本地存储

src/views/Home/

主页组件，显示系统配置信息

快速开始

1

克隆项目

git clone https://github.com/Cong0925/vue3_nodejs_websocket_chatRoom.git
2

启动服务器

cd server && node index.js
3

启动前端

npm run serve
